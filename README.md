# Salon Demo Website

Accessible, fast static landing page demo for Afreen Unisex Salon in Dehradun.

## Contents

- `index.html` - single-page landing page with embedded CSS and LocalBusiness JSON-LD.
- `assets/` - responsive placeholder hero image variants.
- `owner-verification-checklist.md` - items the business owner should confirm before publication.

## Notes

The page intentionally marks unverified services, hours, images, review text, and address formatting as placeholders. Replace those items with owner-approved content before using this as a live production website.
